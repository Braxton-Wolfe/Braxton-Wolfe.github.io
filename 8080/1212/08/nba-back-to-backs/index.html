NBA Back-to-Back Analysis

2025-12-08

Introduction

Back-to-back games are one of the most challenging aspects of the NBA schedule. Teams must play two games in two nights, often with travel in between. This analysis examines how back-to-backs affect team performance across all six NBA divisions.

Division Scoring on Back-to-Backs

# Load combined data
nba_data <- read.csv("/Users/braxtonwolfe/Documents/nba_all_data.csv")

# Process data
nba_data$Date <- ymd(nba_data$Date)
nba_data$Points <- as.numeric(nba_data$Points)

# Calculate B2B vs Rest
nba_data <- nba_data %>%
  arrange(Team_Rank, Date) %>%
  group_by(Team_Rank) %>%
  mutate(Days_Rest = as.numeric(Date - lag(Date))) %>%
  ungroup() %>%
  mutate(Game_Type = ifelse(Days_Rest == 1, "B2B", 
                             ifelse(Days_Rest >= 2, "Rest", NA))) %>%
  filter(!is.na(Game_Type))

# Calculate division averages
summary_data <- nba_data %>%
  group_by(Division, Game_Type) %>%
  summarise(PPG = mean(Points, na.rm = TRUE), .groups = "drop")

# Create chart
ggplot(summary_data, aes(x = Division, y = PPG, fill = Game_Type)) +
  geom_col(position = "dodge", width = 0.7) +
  scale_fill_manual(values = c("B2B" = "#EE5A6F", "Rest" = "#3498DB"),
                    labels = c("B2B" = "2nd Night B2B", "Rest" = "With Rest")) +
  labs(title = "Division Scoring: Back-to-Backs vs Rest",
       y = "Points Per Game", x = "Division", fill = NULL) +
  theme_minimal(base_size = 14) +
  theme(legend.position = "top")

All six divisions score fewer points on the second night of back-to-backs, with some divisions affected more than others. The Pacific division maintains the highest scoring even when tired.

Conclusion

Back-to-back games create measurable performance drops across the NBA. This analysis demonstrates that fatigue affects all divisions, but the impact varies significantly. Schedule balance remains an important consideration for competitive fairness in the league.
